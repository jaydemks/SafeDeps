# Project Media

Use this folder for repository media that is meant to be committed and shown in
GitHub, documentation, package pages, or release notes.

Good examples:

- README cover/banner images
- project icons and logos
- screenshots used by documentation
- small diagrams or UI preview images

Avoid putting generated scan outputs here. Use `security-artifacts/` for local
reports, SBOMs, SARIF files, and other generated security outputs.

Suggested naming:

- `safedeps_banner.png`
- `safedeps_icon.png`
- `ui_scan_results.png`
- `docs_policy_editor.png`
